CSS ia a fantastic tool that allows us to style HTML elements. Because CSS is scripted on a serparate page it allows for quickly and easily making changes across multiple web pages and less coding.

CSS lets us create consistent styling, improved typography, responsive design, and layout techniques like grid and flexbox.
