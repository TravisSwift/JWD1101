Block Elements

A block element stretches from left-to-right and takes up all of the space that it can on the page. p is another example of a block element. When you use a p or a div by itself, nothing can be placed to the right of it and we move down to the next line of our page.

Inline Elements

An inline element only takes up as much space as it needs to. The td table cell is an example of an inline element. We were able to put multiple td side by side from left to right without having to move down to the next line.

span is the most common inline element used in HTML. span is an empty container like div, only it follows the inline rules and does not take up all the remaining space on a page. The height and width of inline elements cannot be adjusted by CSS through normal means.
