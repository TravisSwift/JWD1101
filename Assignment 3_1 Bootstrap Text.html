A List of Reasons To Use Bootstrap

  • Save Time Writing CSS
  • Gives you a Blueprint To Follow
  • Widely Used
  • Helps you avoid making mistakes
  • Achieve difficult effects easily with built in components
  • Good for "Prototyping"
  • You don't have to worry about color schemes and art design

A List of Reasons To Use Custom CSS

  • Deeper Knowledge of CSS
  • BS is "Extra Stuff on Top"
  • BS Makes HTML more difficult to read
  • BS Looks Cookie-Cutter unless you custom modifications
  • BS Changes and updates often, requires relearning
