What is a div?

A div provides us with a way grouping elements together.

What can divs be used for?

Divs can be used to group together different "parts" of our page.

Divs with specific names are called semantic elements

header, nav, section, article, footer are all examples of semantic elements

A list of common semantic elements:

  • article
  • aside
  • details
  • figcaption
  • figure
  • footer
  • header
  • main
  • mark
  • nav
  • section
  • summary
  • time
